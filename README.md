
# Task 3 – SQL Basics (Filtering, Sorting & Aggregations)

## Dataset
Global Superstore / Superstore Dataset (CSV)

## Objective
Practice basic SQL queries including filtering, sorting, aggregations, GROUP BY, HAVING, BETWEEN, and LIKE.

## Queries Performed
1. Counted total records and previewed dataset.
2. Filtered data using WHERE clause.
3. Sorted sales data using ORDER BY.
4. Generated summary reports using SUM, AVG, COUNT.
5. Filtered aggregated results using HAVING.
6. Used BETWEEN for date range analysis.
7. Used LIKE for pattern search on customer names.
8. Identified top 5 customers by total spend.

## Files Included
- queries_task3.sql – All SQL queries used in the task
- sales_summary.csv – Exported summary output (to be generated from SQL)
- README.md – Explanation of queries and approach

## Outcome
Built confidence in SQL fundamentals including data exploration, filtering, aggregation, and reporting.
